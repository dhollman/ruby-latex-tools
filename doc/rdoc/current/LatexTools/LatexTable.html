LatexTools::LatexTable

The LatexTable class represents a table in Latex. It provides multiple means of creating, formatting, and outputting Latex tables.

Basic Examples

The easiest way to create tables programmatically is to use the #<< method. 

  t = LatexTable.new(3)
  t << " " << "Height" << "Weight" << endl
  t << hline
  t << "Person 1" << '156 cm' << '55 kg' << endl
  t << "Person 2" << '183 cm' << '62 kg' << endl
  t << hline
  t.to_s

This produces 

  \begin{table}
    \begin{tabular}{ccc}
      & Height & Weight \\
      \hline
      Person 1 & 156 cm & 55 kg \\
      Person 2 & 183 cm & 62 kg \\
      \hline
    \end{tabular}
  \end{table}

More documentation to come in the future, but hopefully this is enough to get you started.

211:         elsif(row.tag == :cline)

216:         elsif(row.tag == :provisional)
217:           raise RuntimeError, "Programmer needs more coffee: Unhandled provisional" unless row_num == @data.size - 1
218:         else

220:             if(entry)

224:               elsif(content.respond_to?(:to_f) && (content.to_f.to_s == content.to_s || content.to_i.to_s == content.to_s) && !@float_number_format.nil?)
